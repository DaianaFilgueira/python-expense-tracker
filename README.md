# Python Expense Tracker

A simple Python program to track personal expenses.

## Features

- Add expenses by category
- Store expenses in CSV
- View summary by category

## Technologies

- Python
- CSV file handling

## How to Run

Clone the repository:

git clone https://github.com/DaianaFilgueira/python-expense-tracker

Run the program:

python expense_tracker.py

## Example

1. Add expense
2. Show summary
3. Exit

Example summary:

Food: $50  
Transport: $30
